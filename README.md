# HR Analytics & Employee Attrition Dashboard

## Project Overview

This project is a comprehensive **HR Analytics Dashboard** developed in Power BI that analyzes workforce data to identify employee attrition patterns, demographics, compensation structures, and workplace satisfaction metrics. The dashboard provides actionable insights for strategic HR decision-making and employee retention strategies.

## Dataset Summary

- **Total Records**: 1,470 employees (1,233 current, 237 ex-employees)
- **Data Attributes**: 41 comprehensive employee metrics
- **Attrition Rate**: 16.1%
- **Data Format**: Excel (.xlsx) with Power BI visualization
